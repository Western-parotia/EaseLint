# 扫描不出 Kotlin JDK API 的问题

lint 发布基于 org.jetbrains.kotlin:kotlin-stdlib:1.4.31

经测试 只有 基于 1.4.31 打包的lint规则 可以检测 1.4.31的 项目

其他版本就算lint 打包依赖的版本高与项目版本也不行。相同也不行，1.5.31,1.6.21 都测试不行。

* 如何解决？
  使用1.4.31 版本

* 使用7.0 AGP 是否可避免 待验证
